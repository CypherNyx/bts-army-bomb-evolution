# BTS ARMY Bomb Evolution

This is an interactive React web application documenting the evolution of BTS's official light sticks (ARMY Bombs) from 2014 to 2026. Built as a cinematic, scroll-driven infographic with Framer Motion animations. Deployed on GitHub Pages.

## 💜 Live Demo
Link: https://CypherNyx.github.io/bts-army-bomb-evolution/

## ✨ Features
- Cinematic hero section with staggered text reveal animation
- Scroll-driven timeline showcasing all 6 ARMY Bomb versions (2014–2026)
- Full tour history for each version including group and solo tours
- Animated stats counter section
- ARMY purple design system — minimalist, sans-serif, image-forward
- Fully responsive — mobile and desktop
- Accessible: WCAG AA contrast, semantic HTML, keyboard navigation

## 🛠 Tech Stack
| Technology | Purpose |
| :--- | :--- |
| React 18 + Vite | Frontend framework and build tool |
| Framer Motion | Scroll animations and entrance transitions |
| CSS Modules + CSS Custom Properties | Scoped styles and design tokens |
| React Router v6 (HashRouter) | Client-side routing for GitHub Pages |
| Inter (Google Fonts) | Typography |
| gh-pages | Deployment to GitHub Pages |

## 💡 Light Sticks Covered
| Version | Year | Notable Tour |
| :--- | :--- | :--- |
| Original Light Stick | 2014 | The Red Bullet Tour |
| ARMY Bomb Ver. 1 | 2015 | BTS Begins · HYYH On Stage |
| ARMY Bomb Ver. 2 | 2017 | The Wings Tour (first Bluetooth) |
| ARMY Bomb Ver. 3 | 2018 | Love Yourself World Tour |
| Map of the Soul Special Edition | 2020 | MOTS Tour (COVID-cancelled) |
| ARMY Bomb Ver. 4 | 2026 | Arirang World Tour |

## 💻 Getting Started (Local Development)

```bash
# 1. Clone the repo
git clone https://github.com/CypherNyx/bts-army-bomb-evolution.git

# 2. Navigate into the folder
cd bts-army-bomb-evolution

# 3. Install dependencies
npm install

# 4. Start dev server
npm run dev
```
*5. Open browser at [http://localhost:5173](http://localhost:5173)*

## 🚀 Deployment
This application deploys to GitHub Pages using the `gh-pages` package.
To publish a new version to the `gh-pages` branch, simply run:
```bash
npm run deploy
```
Note: `HashRouter` is used instead of `BrowserRouter` for GitHub Pages routing compatibility.
